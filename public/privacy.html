Privacy Policy

This application ("Loan Readiness") is a private tool with a single user (its owner). It is not distributed publicly and does not collect data from, or about, any other individuals.

When connected to QuickBooks Online, the application retrieves financial reports (profit & loss, balance sheet, cash flow, and accounts receivable/payable aging) via the QuickBooks API, using an access token stored locally on the server that runs this application. That data is used only to compute loan-readiness ratios and generate a summary narrative, and is not shared with, or sold to, any third party.

Uploaded CSV files are processed only to generate the same in-session summary and are not retained beyond what's needed to complete that session.

Financial narratives are generated using the Anthropic API; the specific figures sent are limited to the summary values needed to draft that narrative.

Questions about this application's data handling can be directed to its owner directly.
